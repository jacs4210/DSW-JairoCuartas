Atributos y valores

Las etiquetas HTML manejan diferentes atributos con sus respectivos valores para poder garantizar un comportamiento a la página web, ya sea redireccionar un clic, indicar el lugar donde se almacena una imagen, un audio, entre otras. A continuación se definirán algunos atributos y que valores debe de tomar:

  • src: Este atributo le permite al navegador conocer la ubicación de un recurso que debe desplegar, este atributo va asociado a etiquetas como img y el valor que toma es la ruta de la imagen en el servidor, por el cual el navegador automáticamente genera una petición GET / HTTP para poder obtener esta imagen del servidor.
  • style: Permite asignar estilos CSS a una propiedad en especifico de una pagina html, por lo tanto, es posible que adopte valores como "background-color: red;", entre otros. Esto es aplicable a cualquier etiqueta HTML.
  • action: "La URL del programa que procesa la información enviada en el formulario." (Mozilla, 2021)
  • align:strong> "Especifica el alineamiento horizontal del elemento." (Mozilla, 2021)
  • href: "La URL de un recurso asociado." (Mozilla, 2021)
  • id: "Identificador el elemento para acceder al mismo desde CSS o Javascript, este indicador debe ser unico para cada elemento." (Mozilla, 2021)

Por otro lado, el HTML posee 2 caras en cuanto a los atributos, donde los atributos se clasifican en atributos de contenido y atributos IDL.

"El atributo de contenido es aquel que se define desde el contenido (Codigo HTML) y puedes definirlo u obtenerlo con element.setAttribute() o element.getAttribute(). El atributo de contenido es siempre un string incluso cuando el valor esperado en un integer. Por ejemplo, para definir el maxlength de un "input" a 42 usando el atributo de contenido, se debe llamar a setAttribute("maxlength", "42") en ese elemento."(Mozilla, 2021)

"El atributo IDL, tambien conocido como propiedad JavaScript. Estos son atributos que se pueden leer o definir usando JavaScript como element.foo.. El atributo IDL siempre va a usar (pero puede transformarse) en el atributo de contenido subyacente para retornar un valor cuando lo obtiene y lo guarda en el atributo de contenido cuando es definido. En otras palabras el atributo IDL, en esencia, refleja los atributos de contenido."(Mozilla, 2021)
